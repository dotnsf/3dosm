# 3D OSM(3D OpenStreetMap)


## Overview

OSM(OpenStreetMap) 地図を 3D 回転して表示するサンプル


## Specification

- 実際のアプリケーション（ブラウザ）画面の（5 x 5 =）25 倍のデータをロードすることで回転表示時に空白部分ができにくいようにしている。

  - このためパフォーマンスが少し悪い

  - パフォーマンスを上げるには index.html 内の #demoMap のスタイルシートを変更する
